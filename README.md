# NullByte Portfolio 🔐

A professional cybersecurity portfolio website with interactive features and a modern hacker-themed UI.

## 🚀 Features
- Terminal-style interface
- Cybersecurity tools simulation
- Responsive design
- Backend with Node.js

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express

## 📂 Project Structure
- Frontend: public/
- Backend: server.js

## ▶️ Run Locally
1. npm install
2. node server.js
3. Open public/index.html
